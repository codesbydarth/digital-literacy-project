## 📌 Project Overview

This project was completed as part of the Digital Literacy course at VIT Bhopal University. The objective was to understand and apply essential digital skills required for academic, professional, and personal use.

The project is based on the role of a Student Digital Ambassador, where I explored different aspects of digital literacy such as online presence, safe internet usage, communication, and cybercrime awareness.

All tasks have been organised into this repository along with relevant screenshots, documents, and reflections.
All work is documented here and in the written Project Report inside the `report/` folder.
